# 如何构建 APK（免费，无需 Android Studio）

使用 **GitHub Actions** 在云端自动编译，全程点击操作，无需安装任何开发工具。

---

## 第一步：注册 GitHub 账号

1. 打开 https://github.com
2. 点击右上角 **Sign up**，注册免费账号（已有账号直接登录）

---

## 第二步：创建新仓库

1. 登录后点击右上角 **+** → **New repository**
2. Repository name 填写：`ARVideoViewer`
3. 选择 **Public**（免费账号必须选公开）
4. **不要勾选** Initialize this repository
5. 点击 **Create repository**

---

## 第三步：上传项目文件

1. 在新建的仓库页面，点击 **uploading an existing file**（页面中间的蓝色链接）
2. 将解压后的 `ARVideoViewer` 文件夹中 **所有文件和文件夹** 拖拽到上传区域
   - 重要：需要保持目录结构，建议用下面的方式：
   - 点击 **choose your files**，选中所有文件夹内的文件上传
   - **或者**：直接把解压的文件夹整个拖进去（Chrome 支持拖拽文件夹）
3. 滚动到页面底部，点击 **Commit changes**（绿色按钮）

---

## 第四步：等待自动构建

1. 文件上传完成后，点击仓库顶部的 **Actions** 标签
2. 你会看到一条正在运行的任务（黄色圆圈 = 运行中）
3. 等待约 **3–8 分钟**，变成绿色勾 ✓ 表示成功

---

## 第五步：下载 APK

1. 点击绿色的构建任务名称
2. 滚动到页面底部 **Artifacts** 区域
3. 点击 **ARVideoViewer-debug-apk** 下载
4. 解压下载的 zip，里面就是 `app-debug.apk`

---

## 第六步：安装到手机

### 方法 A：用数据线传输
1. 用 USB 线连接手机和电脑
2. 手机上选择 **文件传输** 模式
3. 将 `app-debug.apk` 复制到手机存储
4. 在手机文件管理器中找到 apk，点击安装

### 方法 B：用微信/QQ 传给自己
1. 在 PC 版微信/QQ 给自己发送 `app-debug.apk` 文件
2. 手机上接收并打开安装

### 安装前需要开启"未知来源"
- 小米：设置 → 隐私保护 → 安装未知应用 → 允许
- 华为：设置 → 安全 → 更多设置 → 安装未知应用
- 三星：设置 → 生物识别和安全 → 安装未知应用
- 其他：设置 → 安全 → 未知来源（开启）

---

## 常见问题

**Q：Actions 标签下没有任务？**
A：检查是否上传了 `.github/workflows/build.yml` 文件，注意 `.github` 是以点开头的隐藏文件夹。

**Q：构建失败（红色 ×）？**
A：点击失败的任务查看日志，截图发给我帮你排查。

**Q：手机提示"解析包时出现问题"？**
A：你的手机 Android 版本低于 8.0，暂不支持。

---

## 设备要求

- Android 8.0 及以上
- 有后置摄像头
- 有陀螺仪（绝大多数 2018 年后的手机都有）
